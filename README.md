### 🔎 서비스 개요

전라남도 청년들에게 제공되는 문화복지카드 사용처를 쉽게 검색할 수 있는 웹 서비스 입니다.

키워드로 가맹점을 검색할 수 있는 기능과 현 위치 중심으로 지도를 통해 가맹점을 검색할 수 있는 기능을 제공합니다.

### 📆 개발 기간

-   1차 : 2023.4.20 ~ 2023.5.02
-   2차 : 2023.5.02 ~ 2023.5.25
